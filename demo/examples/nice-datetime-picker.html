Nice date time picker

Command Description Default
model Binding model
format String, datetime format for input label DD.MM.YYYY HH:mm
date Boolean, is datepicker enabled
time Boolean, is timepicker enabled
width String, width of entire dtp-picker 300
enable-ok-buttons Boolean, is ok/cancel buttons enabled false
lang String, which locale to use - you must load angular locales first en
min-date String, minimum date (use: 2011-09-01T13:20:30+03:00 for example)
max-date String, maximum date (use: 2011-09-01T13:20:30+03:00 for example)
week-start String, which day does the week start (0-sunday, 1-monday, ...) 1
ok-text String, text on OK button
cancel-text String, text on CANCEL button
title String, title before input
field-width String, bootstrap classes that defines width of field col-sm-8
label-width String, bootstrap classes that defines width of label col-sm-4 
<nice-datetime-picker
    model="dateTimePicker"
    title="Date picker"
    date="true"
    time="false"
    format="DD.MM.YYYY">
</nice-datetime-picker>
